 # 🔐 Caesar Cipher Encryption Tool

## Project Overview

This project demonstrates the implementation of the classical Caesar Cipher encryption algorithm using both a Python command-line application and an interactive web application.

The main objective of this project is to understand the fundamentals of cryptography by implementing encryption and decryption techniques that shift alphabetic characters using a secret key.

The project includes input validation, encryption, decryption, round-trip testing, and a modern web interface.

This project was completed as a cybersecurity portfolio project to demonstrate practical skills in Python programming, JavaScript development, and basic cryptography concepts.

# Encryption Process

The Caesar Cipher works by shifting each alphabet letter by a fixed number of positions.

Example:

Original Message:

HELLO

Shift Key:

3

Encrypted Message:

KHOOR

Decryption restores the original message.
